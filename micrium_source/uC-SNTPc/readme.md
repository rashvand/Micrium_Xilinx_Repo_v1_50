# uC/SNTPc

µC/SNTPc is Micrium's SNTP (Simple Network Time Protocol) client. It features:

* Delivered with complete 100% ANSI C source code and in-depth documentation.
* Easy configuration
* Simple API

µC/SNTPc is a component of µC/TCP-IP.
